"Brain-games" - these are 5 logical console games in which you need to answer questions.

Minimum requirements:
- Node js.

Installation and startup instructions: 
- clone repository
- make install

Launching games:
- brain-even
- brain-calc
- brain-gcd
- brain-progression
- brain-prime
